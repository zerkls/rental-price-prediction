# Прогнозирование цен на аренду жилья в Нью-Йорке

## Задача
Предсказать стоимость аренды квартиры на основе её характеристик (количество комнат, расположение, описание и т.д.).

## Что сделано
- **EDA и визуализация** - анализ распределений, корреляций, выбросов
- **Feature Engineering** - создание новых признаков (удаленность от центра, ключевые слова в описании, квадратичные признаки)
- **Модели** - линейная регрессия, дерево решений, наивные модели (mean/median)
- **Метрики** - MAE, RMSE

## Результаты
| Модель | MAE (train) | MAE (test) |
| :--- | :--- | :--- |
| Decision Tree | 756 | 954 |
| Linear Regression | 785 | 986 |
| Naive (median) | 1086 | 1323 |

Дерево решений показало лучший результат на тестовых данных.

## Данные

Датасет: [Two Sigma Connect: Rental Listing Inquiries](https://www.kaggle.com/c/two-sigma-connect-rental-listing-inquiries/data)
